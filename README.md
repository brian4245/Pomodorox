# Nall-e
The project is about a circuit that detects movement. It shows a easy system you can use in your life. 

### ¿How I use it?

It is easy to use, only thing you should do is arrange it according to what you want.
But I did it for one main reason: to detect people who want to enter your bedroom or any other private room you have.
